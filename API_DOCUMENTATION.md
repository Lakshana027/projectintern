# ProjectIntern

## Overview

ProjectIntern is a web-based project management system that allows users to manage projects, authenticate users, communicate in real time, and monitor project activities.

## Features

* User Authentication
* Project Management
* Real-Time Communication using WebSockets
* Dashboard Interface
* Database Integration using Prisma
* REST API Support

## Technology Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* NestJS

### Database

* Prisma ORM

## Installation

### Clone Repository

git clone https://github.com/Lakshana027/projectintern.git

### Navigate to Project

cd projectintern

### Install Dependencies

npm install

### Run Application

npm run start

### Development Mode

npm run start:dev

## Deployment

The application is deployed on Vercel.
